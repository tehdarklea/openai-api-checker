Many API providers claim to offer OpenAI models but actually:
- 🔄 **Substitute GPT-3.5 for GPT-4** — You pay for GPT-4 but get GPT-3.5
- 💸 **Inflate token counts** — You're charged for tokens you didn't use
- 🔀 **Route through relay servers** — Your data passes through unknown intermediaries
